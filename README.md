How to build
------------
For building:
	$cd src

For cleaning the project:
	$cd src
	$make clean


How to execute
--------------
  edit
        ./myARMSim test/<filename>.mem
  $make

For Windows
--------------
gcc myARMSim.c main.c -I ../include -w -o ../bin/myARMSim
cd ..\bin
myARMSim add.mem
